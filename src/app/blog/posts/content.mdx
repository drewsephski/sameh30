## Economic Optimization

### Production Economics Analysis

Calculate economic optimization parameters:

```python
import numpy_financial as npf

# Net Present Value calculation
def calculate_npv(cash_flows, discount_rate):
    return npf.npv(discount_rate, cash_flows)

# Economic limit calculation
def economic_limit(production_data, price, costs):
    revenue = production_data * price
    net_cash_flow = revenue - costs
    return calculate_npv(net_cash_flow, 0.12)
```
